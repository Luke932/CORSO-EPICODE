Con il tag abbr si definiscono gli acronimi, tipo HTML.
Il tag <code> stampa il carattere monospace tipico degli editor di codice.
Il tag mark evidenzia il testo, il tag q lo racchiude tra virgolette inglesi.

Con il tag <pre> forziamo il browser a leggere spazi e daccapo.
Esempio: 

                Questo testo è rientrato,
                va daccapo,
                legge gli spazi          superiori al primo,
                appare come il carattere monospace
                tipico degli editor di codice.

Con il tag keyboard rappresentiamo un comando da tastiera (es. CTRL + C = copia), con il tag samp rappresentiamo un esempio (es. Finestra di dialogo).

Con il tag var si rappresenta semanticamente una variabile (es. miaVariabile), con i tag sup e sub si rappresentano rispettivamente apice e pedica (es: 28 = 256, H2O formula dell'acqua).
